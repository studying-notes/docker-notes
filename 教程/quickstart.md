## 拉取基础镜像

```shell
docker pull ubuntu:20.04
```

## 在基础镜像上创建容器

```shell
docker create -it --name photoprism ubuntu:20.04
```

## 启动容器

```shell
docker start photoprism
```

## 进入正在运行的容器

```shell
docker exec -it photoprism bash
```
